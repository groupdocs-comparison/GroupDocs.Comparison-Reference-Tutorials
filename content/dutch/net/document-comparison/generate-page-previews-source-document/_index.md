## Stap 2: Voorbeeldopties definiëren
```csharp
PreviewOptions previewOptions = new PreviewOptions(pageNumber =>
{
    var pagePath = Path.Combine(Constants.SamplesPath, $"result_{pageNumber}.png");

});
```

