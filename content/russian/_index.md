- **Что делает GroupDocs.Comparison API?** Он обнаруживает и выделяет изменения между двумя документами одинакового или разного формата.

- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для оценки; коммерческая лицензия требуется для продакшна.

## Что такое GroupDocs.Comparison API?
**GroupDocs.Comparison API** — это набор библиотек, позволяющих разработчикам программно сравнивать содержимое, макет и форматирование документов. Он поддерживает более 100 типов файлов, предоставляет подробные журналы изменений и предлагает варианты принятия или отклонения модификаций через код.

## Frequently Asked Questions

**Q:** Можно ли использовать GroupDocs.Comparison API в коммерческом продукте?  
**A:** Да, для продакшн‑развёртываний требуется действующая коммерческая лицензия. Бесплатная пробная версия доступна для оценки.

**Q:** Поддерживает ли API файлы, защищённые паролем?  
**A:** Абсолютно. Вы можете передать пароль документа при загрузке исходных файлов.

**Q:** Какие версии .NET совместимы?  
**A:** API работает с .NET Framework 4.5+, .NET Core 3.1+, .NET 5 и .NET 6+.

**Q:** Как API обрабатывает большие документы или массовые сравнения папок?  
**A:** Он использует потоковую передачу и оптимизированные алгоритмы для снижения потребления памяти, а также позволяет сравнивать целые каталоги с помощью функции сравнения папок.

**Q:** Можно ли настроить визуальный стиль вывода сравнения?  
**A:** Да, параметры Comparison Options позволяют задавать цвета, стили разметки и форматы вывода для генерируемого диффа.
