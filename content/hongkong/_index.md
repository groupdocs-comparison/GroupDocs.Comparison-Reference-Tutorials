# GroupDocs.Comparison API 教學與開發者指南

![GroupDocs.Comparison Banner](./groupdocs-comparison-net.svg)

歡迎使用 **完整的文件比較指南**，搭配 **GroupDocs.Comparison API**！我們的完整教學將示範如何有效辨識不同格式文件之間的差異，涵蓋 **Word、PDF、Excel、PowerPoint、圖片等**。無論您是建立 .NET 網路服務或 Java 桌面應用程式，本指南都提供快速整合強大文件比較功能的實用步驟。

## 快速解答
- **GroupDocs.Comparison API 的功能是什麼？** 它會偵測並標示兩份相同或不同格式文件之間的變更。  
- **支援哪些平台？** .NET（Framework、.NET Core、.NET 5/6）以及 Java（8+）。  
- **開發是否需要授權？** 可使用免費試用版進行評估；正式上線需購買商業授權。  
- **可以比較受密碼保護的檔案嗎？** 可以——API 支援提供密碼以開啟受保護的文件。  
- **能產生視覺預覽嗎？** 當然可以，API 能產生並排或疊加的比較結果預覽圖像。  
- **如何比較整個資料夾？** 使用資料夾比較功能一次處理多個檔案，適合批次驗證。  

## GroupDocs.Comparison API 是什麼？
**GroupDocs.Comparison API** 是一套讓開發者以程式方式比較文件內容、版面與格式的函式庫。它支援超過 100 種檔案類型，提供詳細的變更日誌，並可透過程式碼接受或拒絕修改。

## 為何使用 GroupDocs.Comparison API？
- **多格式支援** – 可直接比較 Word、PDF、Excel、PowerPoint、圖片、電子郵件等多種格式，無需先轉檔。  
- **豐富的變更偵測** – 自動標示插入、刪除、格式調整與樣式變更。  
- **程式化變更管理** – 在工作流程中接受或拒絕特定變更，適合審閱系統。  
- **安全處理** – 安全地處理加密或受密碼保護的文件。  
- **高效能** – 經過最佳化的演算法能有效處理大型檔案與批量資料夾比較。

## 為何使用 GroupDocs.Comparison？
- **多格式支援** – 比較不同格式的文件。  
- **詳細的變更偵測** – 識別插入、刪除、格式變更與樣式修改。  
- **變更管理** – 以程式方式接受或拒絕特定變更。  
- **安全比較** – 處理受密碼保護的文件。  
- **高效能** – 使用最佳化演算法有效比較大型文件。

## 支援的文件格式

GroupDocs.Comparison 支援多種文件格式：

| Category | Formats |
|----------|---------|
| **文字處理** | DOCX, DOC, ODT, RTF, TXT |
| **試算表** | XLSX, XLS, ODS, CSV |
| **簡報** | PPTX, PPT, ODP |
| **PDF 文件** | PDF, PDF/A |
| **圖片** | JPG, PNG, BMP, GIF, TIFF |
| **電子郵件** | EML, MSG |
| **以及其他更多…** | HTML, EPUB, DJVU |

## 常見問題

**Q:** 我可以在商業產品中使用 GroupDocs.Comparison API 嗎？  
**A:** 可以，正式上線需購買有效的商業授權。可使用免費試用版進行評估。

**Q:** API 是否支援受密碼保護的檔案？  
**A:** 當然支援。載入來源檔案時可提供文件密碼。

**Q:** 相容的 .NET 版本有哪些？  
**A:** API 支援 .NET Framework 4.5 以上、.NET Core 3.1 以上、.NET 5 以及 .NET 6 以上。

**Q:** API 如何處理大型文件或批量資料夾比較？  
**A:** 它使用串流與最佳化演算法降低記憶體使用，且可透過資料夾比較功能一次比較整個目錄。

**Q:** 是否可以自訂比較輸出的視覺樣式？  
**A:** 可以，Comparison Options 允許您設定顏色、標記樣式與輸出格式，以產生差異圖。
