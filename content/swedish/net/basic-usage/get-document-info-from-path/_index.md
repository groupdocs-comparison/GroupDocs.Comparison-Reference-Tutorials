```csharp
using (Comparer comparer = new Comparer("SOURCE.docx"))

```

## Steg 2: Hämta dokumentinformation
```csharp

```
 Använda`GetDocumentInfo()` metod för`Source` egendom får vi dokumentinformationen, inklusive filtyp, sidantal och storlek.
