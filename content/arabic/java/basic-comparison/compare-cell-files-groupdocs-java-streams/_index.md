## دليل التنفيذ: مقارنة ملفات Excel خطوة بخطوة

### نظرة عامة: ما الذي سنبنيه

سننشئ حلًا يأخذ ملفي Excel كـ `InputStream`s وينتج نتيجة مقارنة تُبرز جميع الاختلافات. فكر فيه كأداة “diff” للجداول — مفيدة للغاية لتتبع التغييرات في مجموعات البيانات، التقارير المالية، أو أي بيانات منظمة.

جمال استخدام الـ streams هو أنك لست مقيدًا بالملفات المحلية. يمكنك مقارنة ملفات Excel من قواعد البيانات، خدمات الويب، أو أي مصدر آخر يمكنه توفير `InputStream`.

### الخطوة 1: تعريف مسارات الملفات

استبدل `YOUR_DOCUMENT_DIRECTORY` و `YOUR_OUTPUT_DIRECTORY` بالمواقع الفعلية التي توجد فيها ملفاتك:

```java
String sourceFilePath = YOUR_DOCUMENT_DIRECTORY + "/SOURCE_CELLS";
String targetFilePath = YOUR_DOCUMENT_DIRECTORY + "/TARGET_CELLS";
String outputFileName = YOUR_OUTPUT_DIRECTORY + "/CompareCellsFromStream_Result";
```

**ملاحظة مهمة**: تأكد من وجود هذه المسارات وأن تطبيق Java لديك يمتلك صلاحيات القراءة/الكتابة. هذه هي مصدر 90 % من مشاكل “لا يعمل”!

### الخطوة 2: تهيئة Input Streams

افتح الـ streams لكلا ملفي Excel. يضمن بناء `try‑with‑resources` إغلاق الـ streams بشكل صحيح (ستشكر الذاكرة لاحقًا):

```java
try (InputStream sourceStream = new FileInputStream(sourceFilePath);
     InputStream targetStream = new FileInputStream(targetFilePath)) {
    // Our comparison code goes here...
}
```

### الخطوة 3: إعداد كائن Comparer

أنشئ مثيل `Comparer` باستخدام الـ source stream. هذا الكائن يتولى كل الأعمال الثقيلة لعملية المقارنة:

```java
try (Comparer comparer = new Comparer(sourceStream)) {
    // Next, we'll add the target stream and compare
}
```

### الخطوة 4: تنفيذ المقارنة

أضف الـ target stream ونفّذ المقارنة. تُحفظ النتيجة في المسار الذي حددته مسبقًا:

```java
comparer.add(targetStream);
final Path resultPath = comparer.compare(new FileOutputStream(outputFileName));
// Your comparison result is now saved at 'outputFileName'
```

وهكذا! لقد قمت الآن ببرمجة **compare excel files java**. سيظهر ملف النتيجة جميع الاختلافات مميزة وملوّنة.
