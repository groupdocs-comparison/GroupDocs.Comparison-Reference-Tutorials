# GroupDocs.Comparison API 教程与开发者指南

![GroupDocs.Comparison 横幅](./groupdocs-comparison-net.svg)

欢迎使用 **GroupDocs.Comparison API** 的 **文档比较完整指南**！我们的综合教程将向您展示如何高效识别各种格式文档之间的差异，包括 **Word、PDF、Excel、PowerPoint、图像等**。无论您是在构建 .NET Web 服务还是 Java 桌面应用程序，本指南都为您提供快速集成强大文档比较功能的实用步骤。

## 快速答案
- **GroupDocs.Comparison API 的作用是什么？** 它检测并突出显示两个相同或不同格式文档之间的更改。  
- **支持哪些平台？** .NET（Framework、.NET Core、.NET 5/6）和 Java（8+）。  
- **开发是否需要许可证？** 免费试用可用于评估；生产环境需要商业许可证。  
- **可以比较受密码保护的文件吗？** 可以——API 接受用于打开受保护文档的密码。  
- **有没有办法生成可视化预览？** 当然，API 可以创建并排或叠加的比较结果预览图像。

## 什么是 GroupDocs.Comparison API？

**GroupDocs.Comparison API** 是一套库，允许开发者以编程方式比较文档的内容、布局和格式。它支持超过 100 种文件类型，提供详细的更改日志，并提供通过代码接受或拒绝修改的选项。

## 为什么使用 GroupDocs.Comparison API？

- **多格式支持** – 在无需先转换文件的情况下比较 Word、PDF、Excel、PowerPoint、图像、电子邮件等多种格式。  
- **丰富的更改检测** – 自动突出显示插入、删除、格式微调和样式更改。  
- **编程式更改管理** – 在工作流中接受或拒绝特定更改，适用于审阅系统。  
- **安全处理** – 安全地处理加密或受密码保护的文档。  
- **高性能** – 优化的算法高效处理大文件和批量文件夹比较。

## 为什么使用 GroupDocs.Comparison？

- **多格式支持** - 跨不同格式比较文档  
- **详细的更改检测** - 识别插入、删除、格式更改和样式修改  
- **更改管理** - 通过编程方式接受或拒绝特定更改  
- **安全比较** - 处理受密码保护的文档  
- **高性能** - 使用优化算法高效比较大型文档

## 常见问题

**问：我可以在商业产品中使用 GroupDocs.Comparison API 吗？**  
答：是的，生产部署需要有效的商业许可证。可使用免费试用进行评估。

**问：API 是否支持受密码保护的文件？**  
答：当然。加载源文件时可以提供文档密码。

**问：兼容哪些 .NET 版本？**  
答：API 支持 .NET Framework 4.5+、.NET Core 3.1+、.NET 5 和 .NET 6+。

**问：API 如何处理大型文档或批量文件夹比较？**  
答：它使用流式处理和优化算法以保持低内存使用，并且可以通过文件夹比较功能比较整个目录。

**问：是否可以自定义比较输出的视觉样式？**  
答：可以，比较选项允许您为生成的差异定义颜色、标记样式和输出格式。
