```java

try (Comparer comparer = new Comparer("source_protected_doc.docx", new LoadOptions("1234"))) {
    // 下一步將在這裡進行...

```

#### 步驟2：新增用於比較的目標文檔

透過指定路徑和密碼來新增您想要比較的目標文件。

```java
// 新增目標文件及其密碼。

```

#### 步驟3：進行比較

執行比較過程並使用 `compare` 方法。

```java
// 進行比較並儲存結果。
```
