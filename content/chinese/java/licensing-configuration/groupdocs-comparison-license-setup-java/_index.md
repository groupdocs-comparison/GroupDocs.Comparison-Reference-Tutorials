#### 步骤 2：创建许可证实例
创建一个实例 `License` 类别，对于申请许可证至关重要：

```java
import com.groupdocs.comparison.license.License;
```

#### 步骤3：设置许可证
使用 `setLicense()` 应用许可证文件路径并解锁所有功能的方法：

