# GroupDocs.Comparison API 튜토리얼 및 개발자 가이드

![GroupDocs.Comparison Banner](./groupdocs-comparison-net.svg)

**문서 비교에 대한 완전한 가이드**에 오신 것을 환영합니다! 우리의 포괄적인 튜토리얼은 **Word, PDF, Excel, PowerPoint, 이미지 등** 다양한 형식의 문서 간 차이를 효율적으로 식별하는 방법을 보여줍니다. .NET 웹 서비스든 Java 데스크톱 애플리케이션이든, 이 가이드는 강력한 문서 비교 기능을 빠르게 통합하는 데 필요한 실용적인 단계를 제공합니다.

## 빠른 답변
- **GroupDocs.Comparison API는 무엇을 하나요?** 두 문서가 동일하거나 다른 형식일 때 변경 사항을 감지하고 강조 표시합니다.  
- **지원되는 플랫폼은 무엇인가요?** .NET (Framework, .NET Core, .NET 5/6) 및 Java (8+).  
- **개발에 라이선스가 필요합니까?** 평가용으로는 무료 체험이 가능하며, 프로덕션에서는 상업용 라이선스가 필요합니다.  
- **비밀번호로 보호된 파일을 비교할 수 있나요?** 예 – API는 보안 문서를 열기 위한 비밀번호를 허용합니다.  
- **시각적 미리보기를 생성할 수 있나요?** 물론입니다. API는 비교 결과의 나란히 또는 오버레이 미리보기 이미지를 생성할 수 있습니다.

## GroupDocs.Comparison API란?

**GroupDocs.Comparison API**는 개발자가 프로그래밍 방식으로 문서의 내용, 레이아웃 및 서식을 비교할 수 있게 해주는 라이브러리 집합입니다. 100개 이상의 파일 형식을 지원하고, 상세한 변경 로그를 제공하며, 코드로 수정 사항을 수락하거나 거부할 수 있는 옵션을 제공합니다.

## 왜 GroupDocs.Comparison API를 사용해야 하나요?

- **다중 형식 지원** – 파일을 먼저 변환하지 않고도 Word, PDF, Excel, PowerPoint, 이미지, 이메일 등 다양한 형식을 비교할 수 있습니다.  
- **풍부한 변경 감지** – 삽입, 삭제, 서식 조정 및 스타일 변경을 자동으로 강조 표시합니다.  
- **프로그래밍 방식 변경 관리** – 워크플로우에서 특정 변경을 수락하거나 거부할 수 있어 검토 시스템에 적합합니다.  
- **안전한 처리** – 암호화되거나 비밀번호로 보호된 문서를 안전하게 작업할 수 있습니다.  
- **고성능** – 최적화된 알고리즘으로 대용량 파일 및 대량 폴더 비교를 효율적으로 처리합니다.

## 왜 GroupDocs.Comparison을 사용하나요?

- **다중 형식 지원** - 서로 다른 형식의 문서를 비교합니다  
- **상세 변경 감지** - 삽입, 삭제, 서식 변경 및 스타일 수정을 식별합니다  
- **변경 관리** - 프로그래밍 방식으로 특정 변경을 수락하거나 거부합니다  
- **보안 비교** - 비밀번호로 보호된 문서를 작업합니다  
- **고성능** - 최적화된 알고리즘으로 대용량 문서를 효율적으로 비교합니다

## GroupDocs.Comparison API의 일반적인 사용 사례

- **법률 문서 검토** – 계약 수정본 간의 변경 사항을 빠르게 강조합니다.  
- **재무 보고** – 게시 전에 Excel 또는 PDF 보고서의 변경을 감지합니다.  
- **콘텐츠 관리 시스템** – 최종 사용자에게 Word 또는 PowerPoint 파일에 대한 시각적 차이 도구를 제공합니다.  
- **자동화된 QA** – CI 파이프라인에서 생성된 PDF를 기준 템플릿과 비교합니다.

## 자주 묻는 질문

**Q: GroupDocs.Comparison API를 상용 제품에 사용할 수 있나요?**  
A: 예, 프로덕션 배포에는 유효한 상용 라이선스가 필요합니다. 평가용으로 무료 체험을 제공합니다.

**Q: API가 비밀번호로 보호된 파일을 지원하나요?**  
A: 물론입니다. 소스 파일을 로드할 때 문서 비밀번호를 제공할 수 있습니다.

**Q: 어떤 .NET 버전과 호환되나요?**  
A: API는 .NET Framework 4.5+, .NET Core 3.1+, .NET 5 및 .NET 6+와 호환됩니다.

**Q: API가 대용량 문서 또는 대량 폴더 비교를 어떻게 처리하나요?**  
A: 스트리밍 및 최적화된 알고리즘을 사용해 메모리 사용량을 낮게 유지하며, 폴더 비교 기능으로 전체 디렉터리를 비교할 수 있습니다.

**Q: 비교 결과의 시각적 스타일을 맞춤 설정할 수 있나요?**  
A: 예, Comparison Options를 사용하면 생성된 차이에 대한 색상, 마크업 스타일 및 출력 형식을 정의할 수 있습니다.
